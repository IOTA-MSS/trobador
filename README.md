# TangleTunes Smart Contract

## Test and Deploy
This project uses Hardhat to test the smart contract functionalities.


- Install project's dependecies
```shell
npm install
```
- Run automated tests
```shell
npx hardhat test
```
- Deploy smart contract
```
npx hardhat run scripts/deploy.js
```
